# CCO - Command-Line Orchestration Tool

**Release Repository**

This repository contains releases and documentation for the CCO (Command-line Claude Orchestra) tool.

## About CCO

CCO is a powerful command-line tool for orchestrating multi-agent development workflows using Claude. It provides a sophisticated system for coordinating specialized AI agents to handle complex development tasks.

## Installation

See [Installation Guide](./docs/installation.md) for detailed installation instructions.

### Quick Install

```bash
# Download latest release for your platform
curl -LO https://github.com/brentley/cco/releases/latest/download/cco-<platform>
chmod +x cco-<platform>
sudo mv cco-<platform> /usr/local/bin/cco
```

## Version Format

CCO uses the VisiQuate versioning format: **YYYY.MM.N**

- **YYYY** - Four-digit year
- **MM** - Month (1-12)
- **N** - Release counter for that month (starts at 1, resets monthly)

**Example:** `v2025.11.19.1` = First release on November 19, 2025
